## ☁️ 雲端版（Streamlit Cloud）

**參賽連結：** https://cc-contest-app-jbmsgjeevoqjxhdb3ylqdm.streamlit.app

**參賽密碼：** `cccontest2026`

> ⚠️ AI 語音轉文字（Whisper）僅支援本地運行。參賽者需自行將錄音轉為文字後貼入「📝 文字稿」框中。

### 參賽流程
1. 輸入密碼進入系統
2. 選擇自己的姓名（若不在列表，手動填入）
3. 選擇主線類型（首通 / 付款當下 / 權益兌換）
4. 上傳錄音音頻（MP3 / WAV / M4A）
5. **將錄音轉為文字後貼入「📝 文字稿」框**
6. 點擊提交，查看 AI 評分和改進建議

### 語音轉文字工具推薦
- 手機備忘錄 App（大部分支援語音轉文字）
- 騰訊文檔「語音速記」
- Zoom / 騰訊會議自動字幕
- Whisper（本地電腦，見下方）

## ⚙️ 評分維度

| 主線 | 評分維度 |
|------|---------|
| 首通 | ①服務墊高 ②需求引導 ④行動閉環（價值認同）⑤抗拒處理 |
| 付款當下 | ②需求引導 ③雙向利益 ④行動閉環（索取行動）⑤抗拒處理 |
| 權益兌換 | ①服務墊高 ②需求引導 ③雙向利益 ④行動閉環 ⑤抗拒處理 |

**⑤抗拒處理**：家長需先表達抗拒（沒時間/不需要/太貴等），老師再化解才算處理。

**④行動閉環（首通專屬）**：引導家長自己說出「敢開口比一百分重要」= 價值認同閉環完成。

**綜合分：** AI評分 × 60% + 管理員評分 × 40%
